# Support

For normal usage questions, open a GitHub issue:

https://github.com/darks0l/darksol-terminal/issues

For private support from the terminal:

```bash
darksol support --subject "Need help" --message "What happened"
```

For security reports, use `SECURITY.md` and email darksol@agentmail.to.

Do not post private keys, seed phrases, API keys, passwords, session tokens, or full wallet export files in public issues.
